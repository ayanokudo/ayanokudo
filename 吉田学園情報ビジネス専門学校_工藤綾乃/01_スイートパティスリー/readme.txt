◆作品について◆

◇使用言語 : C++
◇SDK　    : DirectX9
◇開発期間 : 2020/6～2020/9(三か月)
◇ジャンル : ちょっとおかしなシューティングゲーム

◆動作確認済み環境◆
Windows10
DirectxInput対応のゲームパッド(ジョイスティック)に対応しています。
作品を起動するにはMicrosoft Visual C++ 2015 再頒布可能パッケージが必要です。(https://www.microsoft.com/ja-jp/download/details.aspx?id=53587)

◆操作方法◆

・WASDキー/十字ボタン : 自機が移動します。

・SPACEキー/ショットボタン : 弾を発射します。

・Xキー/Nキー/パワーアップボタン : パワーアップアイテムを持っている状態で押すと
　パワーアップゲージが点灯している内容がパワーアップします。

・Pキー/STARTボタン : ゲーム中に押すことで一時停止(ポーズ)することができます。

ENTERキー/STARTボタン : タイトルやチュートリアルが面の遷移に使用します。
